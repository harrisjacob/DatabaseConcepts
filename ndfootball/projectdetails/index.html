To Stage 2 Information

Description of project: We want to make a database system that contains information the plays run by Notre Dame and its opponents in order to assist the Notre Dame Football Analytics Program's scouting and self-scouting work. This work includes finding plays that meet certain criteria as well as creating scouting reports that contain many different query results.

Usefullness: This project will be very useful as it will automate the creation of scouting reports that are needed on a weekly basis and can take hours to complete. It will also allow users to more efficiently search for plays that meet a specific criteria rather than the clunky system currently in use.

Realness: Our data is the real data currently used by the Notre Dame Analystics Program. It is broken down by play. In order to satisify safety concerns we had to change play names and formations for the class demostration. Here is a sample row containing 1 play.

Functionality:
We can insert/delete/update records in the database both manually or en masse by uploading a .csv file
We have real data in the systemi which we have gather from the Notre Dame football analytics program
We can search the database and print returned results. This will be accomplished with a series of of radio buttons and dropdown menus to help the user form queries

Assumptions:
- There are many attributes listed on the server that, due to spacing, do not appear in the E/R diagram (see the table snippet listed above)
- The only teams that will be in the ND/Opponents set are Notre Dame and the teams that it plays. However, a game will have a primary team and a secondary team. The primary team will be Notre Dame or one of its opponents if Notre Dame is not playing. This is necessary because in order to scout opponents we must look at their other games. (i.e. we use data from Georgia vs. Murray State) but only Georgia will appear in the ND/Opponents set.
- Plays stored in the database are only those relevant to the Notre Dame offense. These plays include when Notre Dame is on offense and when an opponent is on defense.
